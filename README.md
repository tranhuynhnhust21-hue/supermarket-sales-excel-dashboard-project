# Supermarket Sales Dashboard

## Project Overview
This project analyzes supermarket sales data using Microsoft Excel and presents key business metrics through an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization
- Dashboard Design

## Key Performance Indicators (KPIs)
- Total Sales
- Total Orders
- Average Sale Value
- Top Product Category

## Business Insights
- Identified the best-performing product categories.
- Compared sales performance across different branches.
- Analyzed customer purchasing patterns.
- Tracked overall sales trends.
